# set

Store a value in an array at some index.

```block
[""][0] = "item";
```
## Example

Change your mind and make one of your favorite colors green instead of brown.

```blocks
let favColors = ["orange", "blue", "red", "brown", "yellow"];
favColors[3] = "green";
```
